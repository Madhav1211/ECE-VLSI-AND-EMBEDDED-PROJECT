# Digital Thermometer with LCD Display

## Objective
- Design a digital thermometer to measure temperature
- Display temperature using an LCD screen

## Components Used
- Temperature Sensor (LM35 / similar)
- LCD Display
- Microcontroller / Arduino
- Power Supply

## Working Principle
- The temperature sensor detects temperature
- It converts temperature into electrical signal
- Microcontroller reads the signal
- LCD displays the temperature value

## Features
- Accurate temperature measurement
- Fast response time
- Easy-to-read LCD display

## Applications
- Medical temperature measurement
- Food industry
- Industrial monitoring

## Advantages
- More accurate than analog thermometers
- Easy to use
- Portable and reliable

## Conclusion
This project demonstrates an efficient way to measure and display temperature using a digital system with LCD output.
